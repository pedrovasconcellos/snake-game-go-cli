 The snake game that came with the Nokia 2280 was known simply as Snake. Depending on the region and the phone's firmware version, it matched the classic variant of the Snake II series, very similar to the one made famous on the Nokia 3310.

  Features:

  * 🐍 The snake grows when it eats points.
  * 📈 The larger the snake, the greater the difficulty.
  * 💀 The game ends when the snake hits itself or the obstacles, depending
  on the mode.
